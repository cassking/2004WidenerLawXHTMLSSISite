Information Hotline / Emergency Announcements / Class Cancellations: General Policies

Class Cancellations

Policy For Cancelling Day Classes:
In the event that it becomes necessary for classes to be cancelled, the decision to cancel day classes will be made by 6:30am.

Policy For Cancelling Evening Classes:
In the event that it becomes necessary for classes to be cancelled, the decision to cancel evening classes will be made by 2pm.

Announcements / Cancellations for each campus are issued separately:

  • Registrar's Information HOTLINE in Delaware
  • Registrar's Inforamation HOTLINE in Harrisburg

Information Hot Line

The Hot Line provides current need-to-know information on an automated phonemenu system. Call into the Hot Line to check for:

  • Graduation ceremony location changes due to inclement weather
  • Other vital information
