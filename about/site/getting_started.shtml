Getting Started Using Features of the Widener University School of Law website

You may have noticed when you were looking around that we have used a number of different programs to create our various effects. As such you will need to download a couple of plug-ins (programs that read and play files of various formats). The essential plugins are listed below. Just click on the links to go directly to the file you need for your computer platform and browser.

Plug-ins and Scripts Used on this Site

  • The Acrobat Reader plug-in allows browsers on all platforms to view and print Portable Document Format [PDF] files. PDF documents are small in file size and don't require dependent fonts or image files. They have become the format of choice for the Internet.

    The PDF file format offers many advantages, including:
    1. PDF files are compressed to produce smaller file sizes, which reduces file transfer time and required storage space.
    2. PDF files can link to text, graphic, sound, and movie files.
    3. PDF files can be indexed so that you can easily search their contents.
    4. PDF files have a zoom feature allowing you to better evaluate images and maps.


    To view PDF files through your browser you will need the free Adobe Acrobat Reader. Depending on which version of PDF reader you are using, and how it is configured, one of two things will happen when you encounter PDF files online. Either the file will open directly in your browser, or the file will download to your computer and will launch Acrobat Reader.
  • Download the free Adobe Acrobat Reader
 
  • The Quicktime plug-in from Apple Computer, Inc. allows you to view video clips and streaming video/audio directly in your web browser. Many factors determine your computer's ability to play video. In addition to having the proper plug-in, your computer's internet connection, processor speed, sound card and software configurations all play a part.
  • Download the free Quicktime Player
  • Download the free QuickTime Player for Macintosh
  • Download the free QuickTime Player for PC
 
  • Windows Media Player allows viewing of multiple video formats as well as playing of MP3s. If you are a Windows user, you likely already have one version of this software. Macintosh users will probably need to install.
  • Download the free Windows Media Player
  • Download the free Windows Media Player for Macintosh
  • Download the free Windows Media Player for PC
 
  • JavaScript enhances the interactivity of web. Although nearly every web browser made in the last four years has JavaScript capability, it is possible to disable this functionality. Our navigation menus and pop-up windows, for example, are controlled by JavaScript. In order to view this site as intended, we encourage you to enable Javascript.
  • For Internet Explorer 5.0+ (Windows):
    JavaScript controls Windows: Tools => Internet Options => Security => Custom Level => (scroll way down) => Active Scripting => Enable
  • For Internet Explorer 5.0+ (MacOS):
    JavaScript controls Windows: If you are using Mac OS9: Explorer => Preferences
    If you are using Mac OSX: Edit => Preferences
    =>Java => Enable Java => check Alert on exception => check Log Java output => check Log Java exceptions => Click Web Content => Enable scripting
  • For Netscape 6.0+:
    Edit => Preferences => Advanced => Enable JavaScript for Navigator
  • For Mozilla 5.0+:
    JavaScript controls: Edit => Preferences => Advanced => Enable JavaScript for Navigator
  • For Opera 5.0+ (Windows):
    JavaScript controls: File => Preferences => Plugins => Enable JavaScript
  • For Opera 5.0+ (MacOS):
    JavaScript controls: Edit => Preferences => Plugins => Enable JavaScript (on Mac, start with Edit)
  • For AOL:
    Javascript Controls: Settings => Preferences => Organization =>Internet Properties (WWW) =>Security => Custom Level button =>Scripting => Active => Enable => Click OK
  • For Safari (MacOS):
    Javascript Controls: Safari => Preferences => Click Security =>Enable Javascript
  • For Konqueror(Linux):
    JavaScript controls: Settings => Configure => Browser => Java/JavaScript => Enable JavaScript Globally
