Widener University School of Law Accessibility Statement

This is the official accessibility statement for http://www.law.widener.edu/. If you have any questions or comments, feel free to email the webmaster at cir0300@mail.widener.edu.

Standards Compliance

We are striving for WAI WCAG 1.0 compliance as well as compliance to the Federal Government's Section 508 amendment to the Federal Rehabilitation Act.

We meet most all of the Level 'AAA' WCAG guidelines and Section 508 requirements. There are other minor corrections to be made, but the following are the major exceptions and corrections.
WCAG Level 'A' : Most box requirements
WCAG Level 'AA' : Most box requirements and Relative Sizes
WCAG Level 'AAA' : Most box requirements
Section 508 : Most box requirements

There are some structural items we are still addressing.

We test our pages using Bobby/WebXact, Cynthia Says and pages are also viewed using the text-only browser emulator LynxViewer.

Standards Compliance

This site has been authored using modern web standards (XHTML, CSS and ECMAScript- the standard scripting language for web pages, more commonly known as JavaScript). This means that the content of this site should be accessible to users of any browser or web device although the design of the site will only be viewable in a modern web browser.

The site has also been checked against the World Wide Web Consortium’s Web Accessibility Initiative guidelines and all pages should pass the AAA guidelines.

Where possible, all pages within this site are Bobby AAA approved, meaning they with the Web Accessibility Initiative (WAI) Priority 1, 2 and 3 Guidelines. We have reviewed all the guidelines and believe that all these pages are in compliance.

If you notice any glaring accessibility concerns please contact thewebmaster with details and we will make a reasonable attempt to correct it. Accessibility is a work in progress, so please be patient.

Where possible, all pages within this site are Section 508 approved, complying with all of the U.S. Federal Government Section 508 Guidelines.

All pages on this site validate as XHTML 1.0 Strict. To check for XHTML validity of any page, click on the XHTML logo located inside the footer of every page.

All pages on this site use structured semantic markup. H1 tags are used for main titles, H2 tags for subtitles.

Navigation Aids

  1. Access Keys - usage described above.
  2. Skip Navigation Link - Available on all pages within the web site, allows easy access to main content areas.
  3. The home page and all subpages include a search box (access key s) and a link to the main search page .

Links

  1. Many links have title attributes which describe the link in greater detail, unless the text of the link already fully describes the target (such as the headline of an article).
  2. Links are written to make sense out of context.

Images

  1. All content images used in this site include descriptive ALT attributes. Purely decorative graphics include null ALT attributes.
  2. Complex images include LONGDESC attributes or inline descriptions to explain the significance of each image to non-visual readers.

Visual Layout

  1. This site uses cascading style sheets for visual layout.
  2. This site uses only relative font sizes, compatible with the user-specified "text size" option in visual browsers.
  3. If your browser or browsing device does not support stylesheets at all, the content of each page is still readable.
