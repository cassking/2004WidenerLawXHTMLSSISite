Increasing Text Size via Style Switcher

The style switcher in the left-hand sidebar uses CSS, JavaScript, and the DOM to allow you change this documents text size from the browser window.

The style switcher that is in use on this site requires a number of factors to work as intended. You may have arrived at this page because:

1. You followed a link or typed the URI into your browser address bar.

2. You are viewing this site with a text browser or wireless device. The style switcher will not work in these environments, nor is it meant to.

3. You have disabled Javascript/Cookies in your browser. The style switcher requires both these features to be enabled to work correctly.

4. Your current browser does not support the DOM, ( Document Object Model ), and/or Javascript. We respectfully suggest that you may be interested in upgrading to a browser that supports these modern features. Opera 7+, Mozilla, Firebird, IE6.0 and Netscape 7+ all support the DOM, and will enable you to view this, and indeed many web sites, as the designer intended.

Style Switching can deliver accessibility features such as resizing text or delivering a high visibility version of a page for partially sighted readers. They can also be used to simply give a site an alternative colour scheme or layout.

In short, Style Switchers have very practical uses, but are also a lot of fun. So if you feel your browser is getting a little long in the tooth, why not upgrade and experience a much more functional and interesting internet.
